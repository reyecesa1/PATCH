Core Features

Patch is designed for efficiency and ease of use, with powerful capabilities.

Lightweight & Fast

Optimized for minimal resource consumption with rapid response times. Runs efficiently on standard hardware.

Multi-Model Support

Switch between different AI models based on your needs - from fast responses to complex reasoning tasks.

Session Memory

Maintains context across conversations with categorized memory storage (MOOD, GOAL, NOTE, REFLECTION).

Developer Friendly

Simple setup with clear documentation. Easy to extend and customize for your specific use cases.

Cross-Platform

Privacy Focused

Quick Setup

Get Patch AI Console running on your system in minutes.

1

Install Ollama

First, install Ollama on your system to manage AI models.

# On macOS
brew install ollama
# On Linux
curl -fsSL https://ollama.ai/install.sh | sh
2

Download Patch

Download the Patch AI Console script to your preferred directory.

curl -o patch-console.sh https://patch.ai/download/patch-console.sh
3

Run Patch

Make the script executable and run it to start the console.

chmod +x patch-console.sh
./patch-console.sh
4

Start Using

Begin interacting with Patch. Type 'help' to see available commands.

user@patch:~$ help
